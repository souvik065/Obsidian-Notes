
A **regression task** is a type of machine learning problem where the goal is to **predict a continuous numerical value**.

In simple words:

> If the output is a **number**, and that number can take **any value in a range**, it's a regression task.

---

## 🔹 What Does “Continuous Value” Mean?

A continuous value is something like:

- Price: ₹10, ₹10.5, ₹10.55, ₹10.555…
    
- Temperature: 28.3°C
    
- Salary: ₹7,45,350
    
- Height: 172.8 cm
    

These are not categories — they are **measured quantities**.

---

## 🔹 Examples of Regression Tasks

Here are common real-world regression problems:

1. **House Price Prediction**

    - Input: size, location, number of rooms
        
    - Output: price (₹25,00,000)
        
2. **Stock Price Prediction**
    
    - Input: historical data
        
    - Output: tomorrow’s price
        
3. **Salary Prediction**
    
    - Input: experience, skills
        
    - Output: expected salary
        
4. **Temperature Forecasting**
    
    - Input: weather conditions
        
    - Output: temperature value
        
5. **Time Estimation**
    
    - Input: distance, speed
        
    - Output: arrival time
        

---

## 🔹 Regression vs Classification

|Regression|Classification|
|---|---|
|Predicts numbers|Predicts categories|
|Output is continuous|Output is discrete|
|Example: Price prediction|Example: Spam / Not spam|

If output is:

- ₹50,000 → Regression
    
- "Approved" / "Rejected" → Classification

---

## 🔹 Why Is It Called “Regression”?

The term comes from statistics, especially from work by Francis Galton, who observed “regression toward the mean” in inherited traits like height.

Over time, "regression" became the name for predicting numerical relationships between variables.

---

## 🔹 Technical View (Since You’re Learning ML)

In regression, we try to learn a function:
$$
y = f(x)
$$
Where:

- `x` = input features
    
- `y` = continuous output value
    

The model learns weights that minimize an error like:

- Mean Squared Error (MSE)
    
- Mean Absolute Error (MAE)
